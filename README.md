
# Design CTF Team Banner

A modern, interactive React-based team banner component built with Vite, TypeScript, and Tailwind CSS. Features dynamic visual effects including ASCII rain, cursor effects, and responsive design.

## 🎯 Features

- **Interactive Components** - Built with shadcn/ui and Radix UI
- **Visual Effects** - ASCII rain animation, cursor effects, and scratch overlay
- **Responsive Design** - Mobile-friendly interface using Tailwind CSS
- **Modern Stack** - React, TypeScript, Vite, and Tailwind CSS
- **Accessible** - Built with accessibility best practices

## 📋 Prerequisites

- Node.js 16+ or PNPM
- Git

## 🚀 Getting Started

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Start the development server
pnpm dev
```

The application will be available at `http://localhost:5173`

### Production Build

```bash
# Build for production
pnpm build
```

## 📁 Project Structure

- `src/app/` - Main application components
  - `components/` - Reusable UI components
  - `ui/` - shadcn/ui component library
- `src/styles/` - Stylesheet definitions
- `public/` - Static assets

## 📦 Tech Stack

- **Frontend Framework** - React 18
- **Build Tool** - Vite
- **Language** - TypeScript
- **Styling** - Tailwind CSS & PostCSS
- **UI Components** - shadcn/ui, Radix UI
- **Animation** - Motion, Canvas Confetti
- **Package Manager** - PNPM
